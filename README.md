# 📦 Web technology simple Project (Inventory Management System)

## 📌 Description
This is a simple Inventory Management System developed as part of the System Analysis and Design (SAD) project.  
It helps manage products, users, and basic inventory operations.


## 🚀 Features
- User Login System  
- Add / Manage Products  
- Database Connection using PHP  
- Simple and clean UI  


## 🛠️ Technologies Used
- HTML  
- CSS  
- PHP  
- MySQL  
- XAMPP  


## ⚙️ How to Run the Project
1. Install XAMPP  
2. Start Apache and MySQL  
3. Copy project folder to:
   C:/xampp/htdocs/

4. Open browser and go to:   http://localhost/folder name 



## 📁 Project Structure
- `login.html` → Login page  
- `form.html` → Input form  
- `system.php` → Main logic  
- `db_connect.php` → Database connection
